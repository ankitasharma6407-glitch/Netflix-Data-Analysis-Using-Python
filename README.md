#  Netflix Movie Data Analysis using Python

##  Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a Netflix movie dataset using Python. The analysis focuses on understanding movie trends, genre distribution, popularity, ratings, audience engagement, and release patterns through data cleaning, visualization, and statistical analysis.

The project demonstrates practical data analysis skills using Python and presents meaningful business insights from real-world data.

---

##  Objectives

- Clean and preprocess the movie dataset.
- Explore genre distribution and movie release trends.
- Identify the most popular and highest-rated movies.
- Analyze audience engagement through vote counts.
- Study relationships between popularity, ratings, and voting patterns.
- Generate actionable insights using data visualization.

---

##  Tools & Technologies

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

##  Dataset

The dataset contains information about Netflix movies, including:

- Release Year
- Movie Title
- Genre
- Popularity
- Vote Count
- Vote Average

**Dataset File:** `Netflixdb.csv`

---

##  Exploratory Data Analysis (EDA)

The project answers the following business questions:

1. What is the most frequent genre in the dataset?
2. What genre has the highest votes?
3. What movie got the highest popularity? What's its genre?
4. What movie got the lowest popularity? What's its genre?
5. Which year has the most filmed movies?
6. What are the Top 10 Most Popular Movies?
7. What are the Top 10 Highest Rated Movies?
8. Which Genres Have the Highest Average Popularity?
9. Which Genres Have Received the Highest Total Votes?
10. How Many Movies Were Released Each Year?
11. What is the Relationship Between Popularity and Vote Count?
12. What is the Relationship Between Popularity and Ratings?
13. What are the Top 5 Movies in Each Genre Based on Popularity?
14. Which Decade Produced the Most Movies?

---

##  Key Insights

- Drama is the most frequently occurring genre in the dataset.
- **Spider-Man: No Way Home** achieved the highest popularity score.
- Movie production increased significantly over recent years, with the latest decade producing the highest number of films.
- Movies with higher popularity generally attracted more audience votes, indicating stronger viewer engagement.
- Genre-wise analysis revealed clear differences in popularity and audience interest across movie categories.
- Exploratory data analysis helped uncover trends in movie production, ratings, popularity, and audience behavior.

##  Conclusion

This project demonstrates the complete workflow of **Exploratory Data Analysis (EDA)** using Python. It includes data cleaning, preprocessing, visualization, and business insight generation from a real-world movie dataset.
